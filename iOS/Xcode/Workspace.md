# Workspace

**Project 들과 다른 document 들을 그룹으로 묶는 Xcode document.** 

Workspace 는 하나 이상의 Xcode project, 다른 파일들을 포함할 수 있음. Workspace 는 포함된 프로젝트들, target 간의 implicit, explicit 관계를 제공함.
